# Freelancer Business OS — Widget Suite

> Premium embeddable widgets for Notion, built with HTML/CSS/JS.  
> Part of the Freelancer Business OS template.

## ✨ Widgets

### 1. 💰 Rate Simulator (`index.html`)
Calculate your minimum hourly rate based on income goals, expenses, and billable hours.
- Tax Vault showing monthly/annual tax reserves
- Multi-currency (USD, GBP, EUR, CAD, AUD, JPY)
- Multi-language (EN, JA, ES, FR, DE)

**[Live Demo →](https://miyadai0627-wq.github.io/Freelancer-Rate/)**

### 2. 🍅 Pomodoro Timer (`pomodoro.html`)
Focus timer with 25/5/15 minute modes for deep work sessions.
- Animated progress ring with color-coded modes
- Session, hours, and streak tracking
- Sound notification on completion
- Auto-transition between focus and break modes

**[Live Demo →](https://miyadai0627-wq.github.io/Freelancer-Rate/pomodoro.html)**

### 3. 📈 Revenue Progress (`revenue.html`)
Track monthly revenue goals with a beautiful animated donut chart.
- Real-time percentage calculation
- Color tiers: Red (<30%) → Amber (30-60%) → Green (60-100%) → Purple (100%+)
- Daily target calculation based on remaining days
- Milestone progress bar

**[Live Demo →](https://miyadai0627-wq.github.io/Freelancer-Rate/revenue.html)**

## 🚀 Deploy & Embed

### GitHub Pages
Already deployed at: `https://miyadai0627-wq.github.io/Freelancer-Rate/`

### Embed in Notion
1. Type `/embed` in your Notion page
2. Paste the widget URL
3. Resize the embed block (recommended: 500-600px height)
